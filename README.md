# SAI V1

Proyecto frontend estático de SAI — contenido, estilos y scripts para la página.

Instrucciones rápidas:

- Abrir `index.html` o servir con un servidor local (p.ej. `npx serve` o `python -m http.server`).
- Archivos importantes: `index.html`, `about/index.html`, `styles.css`, `script.js`.
